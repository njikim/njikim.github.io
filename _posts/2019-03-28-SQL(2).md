## 4. DML: INSERT, UPDATE, DELETE
```sql
CREATE TABLE tableTest(
	id	int	null  -- 비워놓아도 상관 x 
    name	varchar(10)  -- 이름 길이는 정해지지 x
    regdate	datetime
);

DROP table tableTest;
```
